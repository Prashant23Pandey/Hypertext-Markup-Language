Web Development Terms

HTML
HyperText Markup Language, the standard markup language for documents designed to be displayed in a web browser.
CSS
Cascading Style Sheets, used for describing the presentation of a document written in a markup language like HTML.
JavaScript
A programming language that enables interactive web pages.
Server
A computer program or device that provides functionality for other programs or devices (clients) over a network.
Browser
A software application for retrieving, presenting, and traversing information resources on the World Wide Web.
