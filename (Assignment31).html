Observing Block Elements

This is a DIV element. Even with a defined width, it takes up the full available horizontal space of its parent container because it is a block element. Subsequent elements will appear on a new line.

This is a P element. Like the DIV, it takes up all available width and forces subsequent content onto a new line, demonstrating typical block behavior.

This is a span element (inline) placed after the p element. Notice it is on a new line because the p element is a block element.
